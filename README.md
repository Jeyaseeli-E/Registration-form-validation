# Registration Form Validation

A simple Registration Form built using **HTML, CSS, and JavaScript** with real-time form validation.

## Features

* User-friendly registration form
* Name field validation (cannot be empty)
* Email format validation
* Password validation (minimum 6 characters)
* Real-time validation while typing
* Error messages for invalid inputs
* Submit button remains disabled until all fields are valid
* Success message displayed after successful form submission

## Technologies Used

* HTML5
* CSS3
* JavaScript (DOM Manipulation)

## Learning Outcomes

* DOM Selection (`getElementById`)
* Event Handling (`addEventListener`)
* Real-time Form Validation
* Regular Expressions (Email Validation)
* Form Submission Handling
* Enabling and Disabling Buttons with JavaScript
